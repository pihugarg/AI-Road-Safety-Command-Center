# 🚦 AI-Road-Safety-Command-Center
AI-powered Road Safety Analytics Dashboard built in Microsoft Power BI featuring interactive KPIs, AI-inspired insights, geographic intelligence, custom tooltips and strategic accident analysis.

---

## 📌 Project Overview

The **AI Road Safety Analytics Dashboard** is an interactive Business Intelligence solution built using **Microsoft Power BI** to analyze UK road accident data from **2021–2022**.

The dashboard transforms raw accident records into meaningful insights using interactive visualizations, custom DAX measures, AI-inspired analytics, KPI cards, dynamic recommendations, and custom tooltip pages.

It enables users to:

- 📊 Monitor accident trends and key performance indicators.
- 🔍 Investigate accident causes using AI-inspired analysis.
- 🗺 Identify accident-prone geographical hotspots.
- 📈 Analyze trends and support strategic road safety planning.

---

# 📂 Dashboard Structure

The dashboard consists of **4 interactive pages** and **3 custom tooltip pages**.

## 📄 Page 1 – Executive Overview

**Business Question:** *What happened?*

### Features

- Executive KPI Cards
- Accident Severity Distribution
- Monthly Accident Trend
- Road Type Risk Analysis
- Weather Impact Analysis
- Severity by Area Type
- AI Executive Insights
- AI Safety Assistant
- Road Safety Index

---

## 📄 Page 2 – AI Accident Investigation Center

**Business Question:** *Why are accidents happening?*

### Features

- AI Root Cause Analysis
- Key Influencers Visual
- Weather Risk Distribution
- Decomposition Diagram
- Road & Junction Speed Analysis
- AI Investigation Summary
- AI Safety Alerts

---

## 📄 Page 3 – AI Geographic Intelligence Center

**Business Question:** *Where are accidents happening?*

### Features

- Top Accident Districts
- Highest Fatality Districts
- Road Type by Area
- Severity by Area Type
- Weather Impact by Area
- Geographic Insights
- Dynamic Risk KPIs

---

## 📄 Page 4 – Road Safety Strategy & Trend Analysis

**Business Question:** *What should we do next?*

### Features

- Monthly Accident Trend
- Monthly Accidents vs Casualties
- Severity Comparison (2021 vs 2022)
- Severity Trend Over Time
- Executive Summary
- Strategic Insights
- Road Safety Index
